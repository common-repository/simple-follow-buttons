A simple plugin that enables you to add follow buttons to all of your posts and/or pages.

== Description ==

Simple Follow Buttons does exactly what it says on the tin – adds follow buttons to all of your posts and pages, simply.

This plugin has a dedicated website! Check out <a href="https://simplefollowbuttons.com" target="_blank">simplefollowbuttons.com</a>

You can even upload and use your own custom images if you wish!

Simple Follow Buttons utilises features that, as a WordPress user, you will be familiar with. A self-explanatory administration screen will have you showing your Follow Buttons on your posts and pages in no time!

== Installation ==

1. Upload the 'simple-follow-buttons' folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Navigate to the SFB Admin Panel via ‘Settings’ -> ‘Follow Buttons’ to change the settings as desired.
